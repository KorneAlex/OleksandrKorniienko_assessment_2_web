## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-repo.git
    ```
2. Navigate to the project directory:
    ```bash
    cd OleksandrKorniienko_assessment_2_web/ver_2.2_11.05.25
    ```
